# Foundations — Composition

How elements are arranged in a field so the whole reads as intentional. Medium decides the units (columns,
margins in px vs. picas); the relationships are universal.

## Grid + structure
A grid is a system of relationships, not a cage: **columns** (rhythm + alignment), **modules**, **margins**
(the frame — generous margins read as confident), **gutters**. The Swiss/modular tradition: a consistent
underlying structure lets variation read as deliberate. The grid exists to be *broken knowingly*.

## Alignment + proximity
**Every element should align to something** — a stray edge reads as a mistake. **Proximity encodes
relationship** (the Gestalt cue from [[found-perception]]): group related things, separate unrelated ones
with space, not lines. Most layout cleanup is tightening alignment and fixing proximity, nothing else.

## Balance
Distribution of visual weight across the field: **symmetric** (stable, formal), **asymmetric** (dynamic,
requires counter-weighting a heavy element with space or a smaller dense one), **radial**. Balance is judged
by the eye ([[found-perception]] visual weight), not measured.

## White space + rhythm
**Negative space is an active material**, not leftover. It creates grouping, emphasis, and pace. **Rhythm**
— consistent intervals (a spacing scale) — makes a layout feel composed; irregular spacing reads as
accidental. Generous, *consistent* space is the cheapest way to look intentional.
